# Ball Catcher Game with Character Animation

## Overview
This is a simple interactive game developed using Python and Turtle Graphics. 
The player controls a character to catch falling balls and score points. 
The game demonstrates core programming concepts such as loops, conditional logic, functions, and event handling.

## Features
- Real-time character movement using keyboard input
- Falling object animation using loops
- Score tracking system
- Collision detection logic
- Modular function-based structure

## Technologies Used
- Python
- Turtle Graphics

## Concepts Applied
- Conditional statements
- Loops
- Functions and modular design
- Basic game logic implementation
- Debugging and structured testing
